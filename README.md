# Employee Management System Project
Project updated
## Introduction

The Employee Management System is a web application built using React.js and Node.js for managing employee records. It allows users to perform CRUD (Create, Read, Update, Delete) operations and stores data in a MySQL database.

---

## Features

* Add Employee
* Edit Employee
* Delete Employee
* View Employee List

---

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js
* MySQL

---

## Installation

```bash
git clone https://github.com/shreeyadosi/Employee-Management-System-Project.git
cd Employee-Management-System-Project
```

```bash
# Backend setup
cd Backend
npm install

# Frontend setup
cd ../Frontend
npm install
```

---

## Usage

### Run Backend

```bash
cd Backend
node server.js
```

### Run Frontend

```bash
cd Frontend
npm start
```

---

## Access the Application

Open your browser and go to:
http://localhost:3000
